
## ▶️ How to Run

### Method 1 (Simple)
1. Extract the ZIP
2. Open `index.html` in Chrome / Edge

### Method 2 (VS Code Live Server) 
1. Open folder in VS Code
2. Install **Live Server** extension
3. Right-click `index.html` → **Open with Live Server**

---

## ✨ Features

### 1) Responsive Navbar
- Mobile menu toggle (☰)
- Smooth scroll to sections

### 2) Animated Counters
- Animals rescued
- Successful adoptions
- Meals served
- Monthly stats

### 3) Donate Form (Demo Payment)
- Choose amount (preset dropdown + manual input)
- Select program
- Donate anonymously option
- Generates receipt preview

### 4) Download Receipt
- Downloads as a **TXT file** (works without any library)

### 5) Contact Form
- Demo submission message (no backend)
